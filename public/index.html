Orca

home || helpers || binding

Orca is simple front end frame work. Its aim is to be easy to learn and use. It offers some basic styling for html tags and has a few easy to use classes. It also had a binding framework and some javascript helper methods. orca.binder class is particularly useful for small projects where you want two-way data binding without pulling in a larger framework like React, Vue, or Angular.

Goals

  • Responsive
  • Small size (< 2.6kb)
  • Make smart looking pages easily
  • Support light/dark themes
  • Css classes to allow responsive rows and cols
  • Css classes to center and hide show elements
  • Simple binding frame work in javascript (no node)
  • Hide and show helpers
  • Easy Dialog feature

Setup

Add this css and this js file to you header

<link rel="stylesheet" href="orca.min.css" />
<script src="orca.binder.js"></script>
<script src="orca.js"></script>
