Orca

Orca is a collection of CSS styles to make simple static sites with semantic html. Orca will manage the styling for you. It offers some basic styling for html tags.

Goals

  • Responsive
  • Small size (< 2.6kb)
  • Make smart looking pages easily
  • Support light/dark themes

Setup

Add this css file to you header

<link rel="stylesheet" href="https://orca1.orca-tools.com/orca.min.css" />

Main tag

Add a main tag in the body. This is where all your content goes.

The main tag is styled with a max-width of 1200px and centered on the page. It will enable the page to be easily responsive.

<main>
    ...content...
</main>
