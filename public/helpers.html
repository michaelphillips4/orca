Dialog Box

This is a simple pop up. Use the static javascript methods to open and close. The dialog uses the html Dialog element. You can have as many as you want on a page.

Dialog.open("elementId"); Dialog.close("elementId");

<button type="button" onclick='Dialog.open("dialog1")'>
Dialog Example 1
</button>

<dialog id="dialog1">
<h3>Suspendisse</h3>
<button
onclick='Dialog.close("dialog1")'>
Close
</button>
</dialog>

Hide Show Helper

The hide show helper will hide or show an element by id or by an instance of the element.

show('elementId');
hide('elementId')
